## Our Solution: PhishGuard Vision

A **Chrome extension** using **4 on-device AI APIs (Gemini Nano)** to create a multi-layer safety shield:

| Layer | What It Does | AI API |
|-------|-------------|--------|
| **Phishing Shield** | 3-tier detection: domain → text → visual screenshot analysis | Prompt API (multimodal) |
| **Fee Decoder** | 3-stage pipeline turns 5,000-word T&C into 4 bullet warnings | Summarizer + Rewriter + Prompt |
| **Smart Ad Block** | AI learns ad patterns from DOM — not stale blocklists | Prompt API |
| **Language Bridge** | Warnings in 10 languages for global protection | Translator API |
